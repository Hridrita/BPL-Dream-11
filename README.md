#  BPL-Dream-11

BPL-Dream-11 is a dynamic sports management application that allows users to build their own premier league dream team. The platform focuses on budget management, strategic player selection, and a responsive user interface to provide an engaging experience for cricket enthusiasts.

---

##  Project Overview
I engineered **BPL-Dream-11** as an interactive team-building platform where users can browse a catalog of players, manage a virtual budget, and select their top 11 players. This project highlights the implementation of real-time state synchronization, conditional rendering based on user actions, and complex UI feedback loops.

##  Technical Stack & Tools
The application is built with a focus on speed and modern design principles:
* **Core Technologies:** HTML5, CSS3, JavaScript (ES6+).
* **Framework:** React.js for managing a dynamic, component-based UI.
* **Styling:** Tailwind CSS and DaisyUI for a polished, mobile-responsive layout.
* **Notifications:** React-Toastify for real-time validation alerts.
* **Icons:** FontAwesome / React Icons for intuitive visual navigation.

##  Key Features
* **Virtual Coin System:** Integrated a coin-based economy where users can claim free credits to "buy" players for their team.
* **Strategic Team Selection:** Users can select up to 11 players, with built-in validation to prevent selecting the same player twice.
* **Dynamic Budget Management:** The system automatically deducts player "prices" from the user's total balance and prevents purchases if the balance is insufficient.
* **Toggle-based Interface:** Implemented a smooth toggle system to switch between the "Available Players" shop and the "Selected Players" lineup.
* **Real-time UI Feedback:** Uses toast notifications to inform users of successful selections, duplicate attempts, or budget constraints.
* **Player Removal:** Users can easily modify their team by removing selected players, which instantly updates the roster count.
* **Responsive Newsletter:** Includes a stylish, responsive newsletter subscription section to engage the user base.

##  Key Dependencies
* `react`: For component logic and state management.
* `react-toastify`: For interactive validation messages.
* `tailwindcss`: For rapid and efficient UI styling.
* `daisyui`: For pre-styled, accessible UI components.

##  Local Installation Guide

Follow these steps to set up the project on your local machine:

1. **Clone the Repository:**
   ```bash

2. **Navigate to the Project Directory:**
   cd BPL-Dream-11

3. **Install Dependencies:**
   npm install

4. **Run the Project:**
   npm run dev
